**Avito-tech-android Project**

**Для работы проекта необходимо в local.properties добавить AVITO_TECH_TOKEN=*свой токен***

Используемый стэк:
- Compose
- Material Design
- Coroutines
- Flow
- Dagger Hilt
- Coil
- Navigation Component
- Retrofit
